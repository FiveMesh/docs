FiveMesh is a cloud infrastructure platform for FiveM servers, powered by sCloud.

Use these docs to configure your workspace, register servers, prepare public asset delivery with FiveMesh CDN, enable resource download caching with FiveMesh Cache and run managed PMA-Voice compatible infrastructure with FiveMesh Voice.

## What FiveMesh manages

- **FiveMesh CDN** hosts public server assets such as inventory images, phone media, loading screen files and other files that need stable public URLs.
- **FiveMesh Cache** proxies FiveM resource downloads through a cache URL generated for each registered server.
- **FiveMesh Voice** runs managed PMA-Voice compatible infrastructure for registered servers.
- **Workspaces** separate your personal services from organization-owned services.
- **API keys** let scripts and tools access service operations with scoped permissions.

<Tip>
  FiveMesh is built for infrastructure and operations. The dashboard focuses on setup, delivery, cache state, usage and access management.
</Tip>
