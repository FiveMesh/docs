This guide takes you from a new account to a workspace that can use FiveMesh services.

## Prerequisites

Before you begin, you need:

- A FiveMesh account.
- Active trial or subscription access.
- Your cfx.re server ID if you want to use Cache or server-aware services.
- Access to your server configuration if you plan to enable Cache.

## Get started

<Steps>
  <Step title="Sign in">
    Open the FiveMesh dashboard and sign in. After authentication, FiveMesh sends you to `/dashboard`.
  </Step>

  <Step title="Complete workspace setup">
    Open **Onboarding** if the dashboard asks for setup details.

    Add your community profile and connect a cfx.re server ID if one is ready. You can continue without a server, but Cache setup stays paused until a server is registered.
  </Step>

  <Step title="Check billing access">
    Open **Billing** and confirm the workspace has active trial or subscription access.

    CDN and Cache dashboards stay locked until billing or trial access is active.
  </Step>

  <Step title="Choose the service">
    Open **CDN** to create a public asset workspace, or open **Cache** to enable resource download caching for a registered server.
  </Step>
</Steps>

<Check>
  When setup is complete, the dashboard overview shows service cards for the active workspace and the next common actions.
</Check>
